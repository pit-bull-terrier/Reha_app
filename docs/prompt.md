## 評価基準
- 100点満点で計算
### 評価基準
- 発音の明瞭さ
- リズムとイントネーション


```
脳損傷後の発話リハビリのための、音読練習用の短い日本語の文章を考えてください。
- 文字数は50~100文字程度
- 口の動きを多く使う単語を含めること
- 難しい感じには、ふりがなをつけてください
- 生成する文章は、日常的で前向きな内容にしてください
```
## フロー
```
1. トップ画面
練習  or  成果

- 練習
・LLMが適当な長文を生成する
・ユーザーがそれを読んで録音する
・音声ファイルをアップロード
・音声をテキストに変換して、それを分析評価する
・評価を表示する
また、トップページに戻りたい
- 成果
・過去5回分の練習結果を表形式で表示
・日時、点数、コメント、（グラフ）
```

### 機能
- トップページ
    - 練習か成果を選択する
    - 最初に入力が必要なので、名前を入力してもらう
- 練習を選択した場合(文章練習)
    - LLMが適当な文章を生成する
    - ユーザーは、その文章を録音する
    - 音声ファイルをLLMにつないで、評価する
    - 評価結果を表示（点数、アドバイスなど）

- 成果を選択した場合
    - 過去5回分の結果が表示される
    - 日時、点数、コメントを表示
